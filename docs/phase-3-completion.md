**Features:**

- ‚úÖ **Capture Groups**: $1, $2, etc. for dynamic URL rewriting
- ‚úÖ **Priority System**: Lower numbers processed first (0-999)
- ‚úÖ **Loop Detection**: Detects circular redirects (A‚ÜíB‚ÜíC‚ÜíA)
- ‚úÖ **Chain Validation**: Warns about long redirect chains (>3 hops)
- ‚úÖ **Duplicate Detection**: Identifies conflicting rules
- ‚úÖ **Bulk Operations**: CSV import/export
- ‚úÖ **CLI Validation**: Pre-deployment checking

**Schema Enhancements:**
```typescript
{
```

### Redirect Processing Flow

```
Request ‚Üí Middleware
  ‚Üì
1. Check canonical host
  ‚Üì
2. Remove trailing slashes
  ‚Üì
3. Fetch redirects (cached 5min)
  ‚Üì
4. Sort by priority (asc)
  ‚Üì
5. Find first match:
   - Exact: string equality
   - Wildcard: convert * to (.*)
   - Regex: test with RegExp
  ‚Üì
6. Apply capture groups ($1, $2, etc.)
  ‚Üì
7. Redirect (301/302/307/308)
```

### Validation Algorithm

**Loop Detection:**
```typescript
1. Start at redirect.from
2. Follow redirect.to
3. Track visited URLs
4. If revisited ‚Üí loop detected
5. If external URL ‚Üí stop
6. Max depth: 10 hops
```

**Chain Detection:**
```typescript
- Count redirect hops
- Warn if > 3 hops
- Display full chain: A ‚Üí B ‚Üí C ‚Üí D
```
